Todo List Application

By: l-burrell on github

READ FIRST - To use this application:

Note 1: Once you begin checking tasks off the list the "add task" button will be removed.

Note 2: Refreshing the page will remove all content, take a screenshot if you need this list to be recorded. No data will be saved on your system otherwise.

Note 3: You must type in the task, then press the "add task" button. Empty tasks cannot be submitted.
